Bienvenue sur CYBER Path.

CYBER Path est un jeu de reflexion dont le but est de deplacer des robots vers des cibles avec le moins de coups possible. Les deplecements sont rectilignes et seuls des obstacles permettent de changer de direction. Des le debut de la partie, vous devrez choisir le nombre de joueur, le nombre de manche puis la difficulté de cette partie. A chaque manche, une cible et un robot sont désignées aleatoirement. Le but des joueurs est de trouver, dans le temps donné, un moyen d'atteindre la cible avec le robot avec le moins de mouvement possible. Le jeu se joue de 2 à 8 joeurs en local. Celui avec le plus de point se verra remporter la partie. 

Déroulement du jeu : 
	Dans un premier temps, le jeu vous demandera de saisir le nombre de joueurs, le nombre de manches et la difficulté du niveau. La difficulté permet de modifier le temps de reflexion necessaire. Pour le niveau facile, le temps de reflexion est de 60 secondes tandis qu'il est de 30 secondes pour le niveau moyen et de 15 secondes pour le niveau difficile. 
	A l'aide de ses reponses le jeu va créer une grille et va afficher le robot a deplacé, la cible a atteindre et va lancer le chrono.
	Il ne vous restera plus qu'a reflechir au nombre de mouvement necessaire pour que le robot rejoigne la cible.
	A la fin du compte à rebours, la grille disparais et il ne vous restera plus qu'a entrer le nombre de deplacement necessaire selon vous.
	Si le robot atteint la cible en un nombre de déplacements inférieur à la valeur indiquée initialement, le joueur courant perd 1 point. Certes il a 
atteint la cible mais il a mal prévu le nombre de mouvements.
	



    Pour lancer le jeu :
    1/ Ouvrir le terminal Linux dans le dossier du jeu
    2/ Taper 'make' dans la console pour compiler les fichiers
    3/ Executer le jeu en tapant './cy-ber_path' dans la console
