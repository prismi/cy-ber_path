Bienvenue sur CY-BER Path, le programme de Nassim ALIOUANE, Ilyes MOUDAKKIR, Prismithan ANANTHAKUMAR.


CY-BER Path est un jeu de reflexion dont le but est de deplacer des robots vers des cibles avec le moins de coups possible. Les deplacements sont rectilignes et seuls des obstacles permettent de changer de direction. Le jeu se joue à 4 joueurs, ensuite il vous sera demandé le nombre de manches que vous voulez jouer puis la difficulté de cette partie. A chaque manche, une cible et un robot sont désignées aleatoirement pour chaque jouuer. Le but des joueurs est de trouver, dans le temps donné, un moyen d'atteindre la cible avec le robot avec le moins de mouvement possible. Celui qui gagne le plus de point se verra remporter la partie. 

Déroulement du jeu : 
	Dans un premier temps, le nombre de manches et la difficulté du niveau. La difficulté permet de modifier le temps de reflexion necessaire. Pour le niveau facile, le temps de reflexion est de 60 secondes tandis qu'il est de 30 secondes pour le niveau intermédiaire et de 15 secondes pour le niveau difficile. 
	A l'aide de ses reponses le jeu va créer une grille et va afficher le robot à deplacer, la cible a atteindre et va lancer le chrono.
	Il ne vous restera plus qu'a reflechir au nombre de mouvement necessaire pour que le robot rejoigne la cible.
	A la fin du compte à rebours, la grille disparait et il ne vous restera plus qu'a entrer le nombre de deplacement necessaire selon vous.
	Si le robot atteint la cible en un nombre de déplacements inférieur à la valeur indiquée initialement, le joueur courant perd 1 point. Certes il a atteint la cible mais il a mal prévu le nombre de mouvements.



Pour lancer le jeu :
    1/ Ouvrir le terminal Linux et aller jusqu'à le dossier du jeu
    2/ Taper gcc -o dans la console pour compiler les fichiers
    3/ Executer le jeu en tapant './cy-ber_path' dans la console
